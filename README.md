# TfL Travel Trends Analysis

This project explores travel patterns across the Transport for London (TfL) network using open data from 2007 to 2021.

## Objective

To analyse how commuter activity has changed over time and identify key trends in network usage.

## Key Focus Areas

* Yearly and monthly passenger trends
* Impact of external factors on travel demand
* Identification of under-served areas of the network

## Dataset

Transport for London (TfL) open data (2007–2021)

## Tools Used

* Python (Pandas, Matplotlib, Seaborn)
* Jupyter Notebook

## Outcome

This analysis provides insights into commuter behaviour and highlights how transport demand evolves over time.

## Future Work

* Incorporate post-2021 data for recent trend analysis
* Apply predictive modelling to forecast passenger demand
* Compare trends across different transport modes
